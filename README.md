# Telecom Network Performance Dashboard 📡

## Overview

This project is a Telecom Network Performance Dashboard built using Microsoft Excel.
The dashboard analyzes BTS (Base Transceiver Station) performance data and presents key telecom metrics using charts, KPI cards, and conditional formatting.

---

## Project Objectives

* Monitor telecom network performance
* Analyze signal strength across BTS sites
* Track downtime and data usage
* Visualize network statistics using dashboards
* Practice real-world telecom data analysis

---

## Features

* KPI Cards

  * Total Users Connected
  * Average Signal Strength
  * Total Data Usage
  * Maximum Downtime

* Charts and Visualizations

  * Users Connected per BTS Site
  * Signal Strength Monitoring
  * Regional Data Usage Analysis

* Conditional Formatting

  * Weak signal detection
  * High downtime highlighting

* Smart Table Formatting

* Dashboard Layout Design

## Dataset

The dataset used in this project is a simulated telecom network dataset representing BTS site performance metrics such as:
